- left/right join与(+)    
>（+）操作符只能出现在WHERE子句中，并且不能与OUTER JOIN语法同时使用。    
 当使用（+）操作符执行外连接时，如果在WHERE子句中包含有多个条件，则必须在所有条件中都包含（+）操作符。    
（+）操作符只适用于列，而不能用在表达式上。    
（+）操作符不能与OR和IN操作符一起使用。    
（+）操作符只能用于实现左外连接和右外连接，而不能用于实现完全外连接。
